# RememberMe
A database to save the data of people that I've met.
## Features (Included Data)
1. Name
2. Where you've met
3. Place (that they live)
4. Contact
	1. Number
	2. e-mail
	3. social media accounts
	4. Website
5. Birthdate
6. Age
7. Likes
8. Dislikes
9. Occupation
10. Last Contact
11. When shold I contact next
12. Additional notes
13. Tags
